```css

```


- Основной: `Inter`

- Карточки: `border-radius: 16px`

## Анимации и интерактив

### Scroll reveal

```js
const observer = new IntersectionObserver((entries) => {
  entries.forEach(e => {
    if(e.isIntersecting) {
      e.target.classList.add('visible');
    }
  });
}, { threshold: 0.1 });

document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
```

```css
.reveal {
  opacity: 0;
  transform: translateY(24px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}
.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}
```

### Sticky nav highlight

При скролле подсвечивать активный пункт меню через IntersectionObserver на секциях.

### Кнопка “Присоединиться”

Все кнопки ведут на: `https://tribute.tinkoff.ru` (или заменить на реальную ссылку оплаты).
